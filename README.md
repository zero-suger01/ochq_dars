# NIS Neural Network Trainer

Interactive educational web application for **Namangan International School** 7-sinf "B" guruh Sun'iy Intellekt ochiq darsi.

## Interactive Lessons

1. **Guruhga bo'lish** — 7-sinf "B" guruh o'quvchilar ro'yxati va guruhlarga bo'lish
2. **Training Steps** — Drag & drop neural network training pipeline (Data Preparation → Forward Pass → Loss Calculation → Backward Pass → Optimization)
3. **Data Types** — Flip cards to learn Python data types
4. **Fake or Real?** — Spot the difference between real photos and AI-generated faces

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, CSS Grid, Animations)
- Vanilla JavaScript (Canvas API for neural network visualization)
